# x366-asm-plugin
A basic `.asm` Intellij plugin for Carson's CSCI-366 class
## Features
- Text highlighting, customizable in settings
- Semantics-aware code completion
- Annotations for unresolved labels
- Comment toggling with <kbd>Command + /</kbd>
- Find usages for labels with <kbd>Command + Left Click</kbd>
- Go to declaration for labels with <kbd>Command + Left Click</kbd>
- Code folding for comments and labels
- Automatic indentation for labels
- Suppresses spell check for code
- Smart renaming for labels <kbd>Shift + F6</kbd>
